# Coastonia Technology Services Website (Next.js)

This is a lightweight, client-safe marketing site for **Coastonia Technology Services** built with Next.js (App Router).

## Pages
- `/` Home (includes **Clients** section with anchor `#clients`)
- `/services`
- `/cloud`
- `/current-engagement`
- `/resources`
- `/contact`

## Run locally
```bash
npm install
npm run dev
```

## Deploy (Vercel)
- Import this repo into Vercel
- Set your domain DNS to Vercel (when ready)
